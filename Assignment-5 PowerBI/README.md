# Assignment 5 - Power BI Dashboard

This assignment demonstrates data visualization and business intelligence capabilities using Microsoft Power BI.

## Project Structure

### Data/
Contains the source data files for the Power BI reports:
- `Sales.xlsx` - Sales transaction data
- `Targets.xlsx` - Sales target metrics

### PowerBI files/
Power BI project files:
- `PowerBIFile.pbix` - Main Power BI Desktop file
- `PowerBIFile.pbit` - Power BI template file
- `PowerBIFilepdf.pdf` - Exported PDF version of the report

### Screenshots/
Visual documentation of the dashboard features:
- **Executive Dashboard.png** - High-level executive overview
- **Product Performance.png** - Product-level analytics
- **Sales Data.png** - Sales data visualization
- **Trend Analysis.png** - Time-series trend analysis
- **Model.png** - Data model relationships
- **R Scripts.png** - R script integration
- Interactive demo screenshots showing various filters and drill-downs:
  - Category filtering (Electronics, Office Supplies)
  - Product-specific views (P014)
  - Time-based filtering (Feb 2024, Oct 2024)
  - Geographic filtering (Odisha)

## Key Features

- **Interactive Dashboards**: Multiple dashboard views for different stakeholder needs
- **Data Modeling**: Structured data relationships between sales and target data
- **Advanced Analytics**: Integration with R scripts for enhanced analytics
- **Dynamic Filtering**: Category, product, time, and location-based filters
- **Performance Tracking**: Sales vs. targets comparison and trend analysis

## Dashboard Views

1. **Executive Dashboard** - Strategic overview for leadership
2. **Product Performance** - Detailed product-level metrics
3. **Trend Analysis** - Historical trends and forecasting
4. **Sales Data** - Granular transaction-level insights

## Getting Started

1. Open `PowerBIFile.pbix` in Power BI Desktop
2. Ensure data connections point to the files in the `Data/` folder
3. Refresh data to load the latest information
4. Explore the interactive dashboards

## Technologies Used

- Microsoft Power BI Desktop
- R (for advanced analytics)
- Excel (data source)
